条件概率


例1.5.1设箱中有甲、乙两厂生产的同型产品100件，其质量情况如下

甲厂 乙厂
正品 50 25
次品 20 5

今从箱中任取一件产品，分别求下列事件的概率:
A = "取得甲厂产品"
B = "取得次品"
AB = "取得甲厂产品，且为次品"
B/A = "已知取得甲厂产品下，取得次品"

解：因为试验E是从100件产品中任取一件观察是否为次品，由古典概率知
P(A) = 70/100 = 7/10 P(B) = 25/100 = 1/4 P(AB) = 20/100 = 1/5 则P(B/A) = P(AB)/P(A) = 2/7

例1.5.5据某日气象预报，次日甲城市的降水概率为0.80，与甲城市相距遥远的乙城市降水率为0.60

  1. 求次日甲、乙两城市都降水的概率
  2. 求次日至少有一个城市降水的概率

解：因甲、乙两城市相距遥远，故可以认为两城市降水与否相互独立
记：A = "甲城市次日降水"
B = "乙城市次日降水"

  1. P(AB) = P(A)P(B) = 0.8 x 0.6 = 0.48
  2. P(A+B) = P(A) + P(B) - P(AB) = 0.8 + 0.6 - 0.48 = 0.92

公式:
P(ABC) = P(A)P(B/A)P(C/BA)
