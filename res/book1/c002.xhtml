例1.4.2某保险盒内装有甲、乙两跟保险丝。根据以往经验，当电流超过额定值10％时，甲保险丝 被烧断的概率为0.8，乙保险丝被烧断的概率为0.7，两保险丝同时被烧断的概率为0.5。求下一次电流超过定额 10%时，至少有一根保险丝被烧断的概率。

    解：记 A=“甲保险丝被烧断”
          B=“乙保险丝被烧断”
        于是 A+B = "甲、乙保险丝至少一根被烧断"
            AB = “甲、乙保险是都被烧断”
    根据概率加法法则有
        P(A+B) = P(A) + P(B) - P(AB)
               = 0.8 + 0.7 - 0.5 = 1 必然有一个烧断

重要公式
P(A1 + A2 + A3) = P(A1) + P(A2) + P(A3) - P(A1A2) - P(A1A3) - P(A2A3) + P(A1A2A3)

练习1.4 - 4某城市有50%住户订日报，65%的住户订晚报，85%住户至少订这两种报纸种的一种, 求同事订这两种报纸的用户的百分比？
解: 采用加法公式 P(A+B) = P(A) + P(B) - P(AB)
其中 P(A+B) 是不论相容还是相交，至少订一种的是这个 85%
P(AB) 是两种都订的概率,那么通过公式来推导
0.85 = 0.5 + 0.65 - P(AB) 那么 P(AB) = 0.5 + 0.65 - 0.85 = 0.3

练习1.4 - 5据统计，某校学生在暑假期间爱好游泳的学生有40%，爱好爬山的有20%。两者 都爱好的有10%，求该校学生暑假期间，对游泳、爬山至少爱好一项的百分比？
解：记 A = "爱好游泳的"
B = "爱好爬山的"
AB = "两者都爱好的"
那么 P(A+B) = P(A) + P(B) - P(AB) = 40% + 20% - 10% = 50%
